## 📝 Описание проекта

Приложение для генерации случайных задач по трём категориям: **Учёба**, **Спорт** и **Работа**. 
Использует библиотеку `random` для случайного выбора задачи из предопределённого списка. 
Ведёт историю всех сгенерированных задач с возможностью фильтрации и сохранения в JSON.

## ✨ Функциональность

- ✅ **Генерация случайной задачи** — выбор случайной категории и задачи из неё
- ✅ **Предопределённый список задач** — 7 задач в каждой категории
- ✅ **История генераций** — все сгенерированные задачи сохраняются с датой и временем
- ✅ **Фильтрация по категории** — просмотр задач только выбранной категории
- ✅ **Сохранение в JSON** — автоматическое сохранение истории при каждом действии
- ✅ **Экспорт истории** — возможность сохранить текущую историю в отдельный JSON-файл
- ✅ **Очистка истории** — с подтверждением пользователя
- ✅ **Валидация данных** — корректная обработка ошибок при загрузке
- ✅ **Визуальные эффекты** — мигание кнопки при генерации
