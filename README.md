# TaskMaster — менеджер задач

**Автор:** [Roman Muslimov]

## Описание
TaskMaster — это графическое приложение для управления задачами с возможностью добавления, фильтрации, сохранения и загрузки данных в формате JSON.

## Функциональность
- ✅ Добавление задач (название, описание, приоритет, дата)
- ✅ Валидация ввода (проверка даты, обязательных полей)
- ✅ Фильтрация задач по приоритету
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Автоматическое сохранение в JSON
- ✅ Загрузка задач при запуске

## Требования
- Python 3.7+
- Tkinter (входит в стандартную библиотеку Python)
